Инструкция по сборке проекта

Для создания сервера с базой данных PostgreSQL использовался Openserver.
Нужно выполнить в следующем порядке:
1. В папке проекта в командной строке нужно ввести следующую команду для подтягивания всех нужных библиотек.
composer install
2. Переименовать файл env в .env и поменять данные для входа в базу.
3. Выполнить следующую команду для создания новой базы и нужных табличек.
php setup-database.php
4. Выполнить следующую команду для запуска сервера.
php spark serve
